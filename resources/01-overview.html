Anthropic API Call Map

Every LLM interaction in the codebase — 3 distinct API call patterns across 2 chat modes + memory system

1:1 Persona Chat
POST /api/chat
Single persona conversation with sliding-window context and per-persona temperature. Streams response tokens back to the browser via SSE.
Anthropic API Call
anthropic.messages.stream()

Model: claude-sonnet-4-20250514
Max tokens: 4096
System: persona prompt + memory summary
Messages: sliding window (~71,904 tokens)
Temperature: 0.3–0.9 per persona
Streaming May trigger memory
Room Group Chat
POST /api/room/chat
Multi-persona group discussion. Up to 3 personas respond sequentially, each seeing prior responses. Optional follow-up round (50% chance).
Anthropic API Calls (per turn)
anthropic.messages.stream() × 1–4+ calls

Model: claude-sonnet-4-20250514
Max tokens: 300
System: room-specific prompt with rules
Messages: room history (~60K tokens)
Temperature: 0.3–0.9 per persona

Each persona is a separate API call. Forced retry if all pass.
Streaming × N
Memory Summarization
Background / POST /api/memory
Summarizes trimmed conversation history into a persistent memory. Triggered automatically when >20 messages are trimmed, or manually via API.
Anthropic API Call
client.messages.create()

Model: claude-sonnet-4-20250514
Max tokens: 1024
System: none
Messages: single user message with
  → summarization prompt template
  → existing memory (if any)
  → trimmed messages text
Synchronous Background (fire-and-forget)
